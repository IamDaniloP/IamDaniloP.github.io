Perfil

Olá, me chamo Danilo e sou fã de tecnologias desde minha adolescência. No momento, curso o 3° período de engenharia da computação e sinto que fiz a melhor escolha para meu futuro. Compartilhar experiências e trocá-las, pesquisar tecnologias novas e estudá-las é algo que sempre busco praticar diariamente. Atualmente, estou focado em tecnologias para aprimorar o desenvolvimento Front-End.

Página Home - Spotify

Página Home do Spotify. Foi desenvolvida com o acompanhamento do professor do curso. Além de aprender mais algumas novas propriedades do CSS3, também foi aplicada novas propriedades do BootStrap4. Essa página conta com alguns ajustes feitos por mim, para uma melhor aplicação aos dispositivos móveis e, também, uma melhor experiência para o usuário.

Página Home - Finans

O projeto foi feito buscando aprimorar os conhecimentos em HTML5 e CSS3. O site foi construido em sua maior parte utilizando BootStrap4 e, quando necessário, para uma melhor compatibilidade com dispositivos móveis, foi utilizado Media Queries.

Jogo - Mata Mosquito

O foco do desenvolvimento do aplicativo era implementar algumas lógicas em JavaScript adquiridas no curso. DOM, BOM, Callback, Funções, Eventos são alguns dos itens presentes no projeto. Como o foco era apenas aprimorar e praticar o JavaScript, não foi priorizado a identidade visual do jogo. Os repositórios contam com descrição para saber cada parte executada no programa. Priorizei sempre que possível transformar linhas de códigos em funções eficientes para deixar o código mais legível e menos sucetíveis à erros.
